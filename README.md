
# MobileAppSwitchForensics
An automated forensic analysis tool for iOS and Android app switching artifacts.  
This project extracts, parses, and correlates data from:
- iOS: KnowledgeC.db, applicationState.db, snapshot (.ktx)
- Android: usagestats, recent_tasks, snapshots

The tool generates structured SQLite and HTML reports for cross-artifact verification of user activity timelines.

---

## Features
✅ Cross-platform (Android & iOS)  
✅ Automatic parsing and report generation  
✅ SQLite export for timeline analysis  
✅ Focused on app-switching and user-behavior verification
